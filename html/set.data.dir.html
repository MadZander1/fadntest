set.data.dir {fadnUtils}R Documentation

Sets the data.dir

Description

Sets the data.dir

Usage

set.data.dir(new.data.dir)

Arguments

new.data.dir

the full path to the folder where the data.dir will be. Ending slash "/" shall not be present

Value

TRUE if succesfully set the data.dir; FALSE otherwise. Returns in invisible mode.
