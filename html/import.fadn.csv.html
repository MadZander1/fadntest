import.fadn.csv {fadnUtils}R Documentation

Imports a DG-AGRI csv into fadnUtils

Description

It first call the convert.to.fadn.raw.rds and then convert.to.fadn.str.rds

Usage

import.fadn.csv(
  file.path,
  raw.f = NULL,
  sepS = ",",
  fadn.year = NA,
  fadn.country = NA,
  keep.csv = F
)

Arguments

file.path

the full path of the file (the filename must be included)

raw.f

the raw_str_map file to use. it must reside inside 'raw_str_maps; folder of the data.dir

sepS

the separator of the csv files (by default ",")

fadn.year

the year the csv files refers to (e.g. 2001)

fadn.country

the three letter country code the csv files refers to (e.g. "ELL")

keep.csv

if TRUE, copy the csv files; else do not copy
