get.rds.stored.data {fadnUtils}R Documentation

Shows a table of the rds.stored.data

Description

the table contains the following fields: TYPE=raw.rds or str.rds NAME=the name of the stored variable DESCRIPTION=a short description of the contents

Usage

get.rds.stored.data()

Arguments

print.also

if T, prints the table of stored data too

Value

a table of stored data
