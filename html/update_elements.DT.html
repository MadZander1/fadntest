update_elements.DT {fadnUtils}R Documentation

Updates selected elements of data stored in one DT with new one given in melted format

Description

The user provides the data.new: id,variable,new value. The function overwrites all existing id-column with the new values

Usage

update_elements.DT(data.old, data.new)

Arguments

data.old

The DT to update

data.new

The data to insert. It must have three columns: id,variable,new value. E.g. data.new=data.table("id"=c(810001100105),"variable"=c("AASBIO_CV"),value=c(999999))

Value

a DT with the updated values
